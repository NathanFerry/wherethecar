# WhereTheCar Project

## About the project
WhereTheCar is a class project. The goal is to create a JavaFX application that manages a fleet of vehicles for 
professionals, with accent on code quality, design patterns, and best practices.

### In depth information
In case you need some precise information, please look at the [Project Documentation](./docs/documentation.md).

### Contributing
This project follow the [MVCI](https://www.pragmaticcoding.ca/javafx/mvci/) pattern from [PragmaticalCoding](https://www.pragmaticcoding.ca/).  
If you want to contribute to the project, please read the article and in order to be familiar with the architecture.
